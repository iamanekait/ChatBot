Chatbot Web App

This is a simple web application that implements a chatbot using OpenAI's GPT-3.5 model. Users can interact with the chatbot by typing messages, and the chatbot will respond accordingly.

Features

- Allows users to type messages and receive responses from the chatbot.
- Uses OpenAI's GPT-3.5 model for natural language processing.
- Provides a simple and intuitive user interface for chatting.

Installation

1. Clone the repository:

   git clone https://github.com/yourusername/chatbot-web-app.git

2. Install the required dependencies:

   pip install -r requirements.txt

3. Set your OpenAI API key:
   
   Replace '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' in app.py with your actual OpenAI API key.

4. Run the Flask server:

   python app.py

5. Open your web browser and navigate to http://localhost:5000 to access the chatbot interface.

Usage

- Type a message in the input field and press "Send" to send it to the chatbot.
- The chatbot will respond with an appropriate message.
- Conversations are displayed in the chat area, with user messages aligned to the right and chatbot messages aligned to the left.
